### 0.3 用户真实需求（来自实际材料分析）

**用户学习风格**：反复输入/默写来记忆（类似五十音图默写训练），不是翻卡片看一眼。

**实际材料格式**：

```
材料 1：必背词条搭配（Excel，658 行）
  格式：A列中文 | B列英文
  类型：词条（国内生产总值 | GDP）
        句式（就业是民生之本。 | Employment is crucial...）
  特点：短小精悍，适合快速反复训练

材料 2：米恩翻译入门 5 篇（Word，5 篇章 103 段）
  格式：先 4 段中文，再 4 段英文（非逐句对照）
  类型：完整翻译段落（100-200 字/段）
  特点：长段落，适合翻译练习和整体记忆
  主题：国际合作、国内政策、国内发展、消费、智能体
```

**核心洞察**：
1. 用户需要**输入型训练**（默写/填空/翻译），不是被动翻看
2. 材料格式多样（Excel 词条、Word 段落），导入必须智能识别
3. **反复循环**是关键——同一批材料要能快速重复训练，不需要每次重新选择
4. 词条和段落是两种不同粒度，需要不同的训练方式

### Sprint 7：材料导入 + 卡片生成（5 天）

#### 7.1 智能材料导入

**现状**：只能手动逐条创建卡片，无法批量导入。

**目标**：支持用户实际使用的材料格式，一键导入。

```
支持的导入格式：
  ├── Excel (.xlsx)
  │   ├── 自动识别列结构（中文列 + 英文列）
  │   ├── 支持 2 列（中|英）和多列（中|英|分类|备注）
  │   ├── 自动分类：词条（< 30 字）vs 句式（≥ 30 字）
  │   └── 示例：必背词条搭配 658 行 → 658 张卡片
  │
  ├── Word (.docx)
  │   ├── 自动识别中英文段落
  │   ├── 智能配对：连续中文段 → 连续英文段
  │   ├── 按篇章标题自动分组（篇章一/篇章二...）
  │   ├── 长段落自动拆分为句子级卡片（可选）
  │   └── 示例：米恩翻译 5 篇 → 按篇章分组的卡片集
  │
  ├── 纯文本 (.txt)
  │   ├── 支持分隔符格式（Tab / | / 换行对）
  │   └── 支持中英交替格式
  │
  └── AI 智能拆分
      ├── 粘贴任意文本 → AI 自动识别中英文
      ├── 自动生成卡片对
      └── 支持用户校对和编辑
```

#### 7.2 卡片集（Deck）概念

**现状**：所有卡片平铺在一个列表里，无法分组管理。

**目标**：引入卡片集，按学习材料组织。

```
卡片集 (Deck)：
  ├── 名称：必背词条搭配
  ├── 来源：必背词条搭配.xlsx
  ├── 卡片数：658
  ├── 类型：词条 + 句式（自动分类）
  ├── 进度：已掌握 120 / 总计 658
  ├── 创建时间：2026-04-20
  └── 标签：翻译、MTI、政治经济

卡片集 (Deck)：
  ├── 名称：米恩翻译 - 国际合作
  ├── 来源：入门积累5篇.docx → 篇章一
  ├── 卡片数：8（4 中 + 4 英，配对后 4 对）
  ├── 类型：段落翻译
  ├── 进度：已掌握 2 / 总计 4
  └── 标签：翻译、国际合作
```

#### 7.3 后端 API

```
POST /api/v1/memory/import/excel     上传 Excel → 返回预览 + 卡片预生成
POST /api/v1/memory/import/docx      上传 Word → 返回预览 + 卡片预生成
POST /api/v1/memory/import/text      粘贴文本 → AI 拆分 → 返回预览
POST /api/v1/memory/import/confirm   确认导入 → 写入数据库

GET  /api/v1/memory/decks             卡片集列表
POST /api/v1/memory/decks             创建卡片集
GET  /api/v1/memory/decks/:id         卡片集详情 + 卡片列表
PUT  /api/v1/memory/decks/:id         更新卡片集
DELETE /api/v1/memory/decks/:id       删除卡片集（含所有卡片）
```

### Sprint 8：输入型训练模式（7 天）

> 核心差异化功能——不是翻卡片，而是**反复默写**

#### 8.1 训练模式设计

基于用户"反复输入默写"的学习风格，设计 4 种核心训练模式：

**模式 1：英→中默写（最常用）**

```
┌─────────────────────────────────────────┐
│  必背词条搭配 · 第 23/658               │
│  ████████████░░░░░░░░  3.5%            │
├─────────────────────────────────────────┤
│                                         │
│  gross domestic product (GDP)           │
│                                         │
│  请输入中文翻译：                         │
│  ┌─────────────────────────────────┐   │
│  │ 国内生产总值                      │   │
│  └─────────────────────────────────┘   │
│                                         │
│  [提交]  [跳过]  [提示]                  │
│                                         │
└─────────────────────────────────────────┘

提交后：
  ✅ 完全正确！+10 分
  或
  ❌ 正确答案：国内生产总值
     你的答案：国民生产总值（差 1 个字）
     [标记已掌握] [稍后再练]
```

**模式 2：中→英默写**

```
┌─────────────────────────────────────────┐
│  请将以下中文翻译为英文：                   │
│                                         │
│  国内生产总值                             │
│                                         │
│  ┌─────────────────────────────────┐   │
│  │ gross domestic product (GDP)     │   │
│  └─────────────────────────────────┘   │
│                                         │
└─────────────────────────────────────────┘
```

**模式 3：填空训练（智能挖空）**

```
┌─────────────────────────────────────────┐
│  填入空缺部分：                           │
│                                         │
│  [        ] is crucial to the people's   │
│  wellbeing.                             │
│                                         │
│  提示：就业是___。                        │
│                                         │

│  │ Employment                       │   │
│  └─────────────────────────────────┘   │
│                                         │
└─────────────────────────────────────────┘
```

**模式 4：段落翻译（针对 Word 长段落材料）**

```
┌─────────────────────────────────────────┐

├─────────────────────────────────────────┤

│  请输入英文翻译：                         │
│  ┌─────────────────────────────────┐   │
│  │ We actively participate in...    │   │
│  │                                  │   │
│  │                                  │   │
│  └─────────────────────────────────┘   │
│                                         │

查看参考译文后：
```

#### 8.2 训练流程

```
选择卡片集 → 选择训练模式 → 开始训练
  │
  ├─ 显示题目（英文/中文/填空/段落）
  ├─ 用户输入答案
  ├─ 提交 → 即时判定
  │   ├─ 正确 → 绿色反馈 + 加分 + 下一题
  │   ├─ 部分正确 → 黄色反馈 + 显示差异 + 可重试
  │   └─ 错误 → 红色反馈 + 显示正确答案 + 加入错题本
  │
  ├─ 训练结束 → 统计面板
  │   ├─ 正确率、用时、连击记录
  │   ├─ 错题列表（可一键加入复习队列）
  │   └─ 薄弱项分析
  │
  └─ 错题本 → 针对性复习
      ├─ 只练错题
      ├─ 间隔复习（SM-2）
      └─ 全部掌握后从错题本移除
```

#### 8.3 连击系统 + 游戏化

```
连击 (Streak)：
  ├── 连续正确 +1 连击
  ├── 连击 ×3 / ×5 / ×10 有特效反馈
  ├── 连击 ×5 解锁"极速模式"（无提示）
  └── 断连击有轻微惩罚感（屏幕微震 + 音效）

积分系统：
  ├── 正确 +10 分
  ├── 首次正确 +5 额外分
  ├── 连击加成：×1.0 → ×1.5 → ×2.0 → ×2.5
  ├── 段落翻译：按 AI 评分比例给分
  └── 每日积分排行（自己 vs 历史）

成就系统：
  ├── 🏅 初学者：完成第一轮训练
  ├── 🔥 连击王：单次训练连击 ×20
  ├── 📚 博学家：掌握 100 张卡片
  ├── 🎯 神射手：正确率 95% 以上（≥ 20 题）
  └── ⚡ 闪电侠：平均每题 < 5 秒
```

#### 8.4 AI 自动出题（背书匠模式）

导入材料后，LLM 围绕材料内容自动生成多种题型，辅助理解和记忆。
不只是"拆卡片"，而是**围绕内容考你**。

```

  ├── 卡片集详情页 → [AI 出题] 按钮
  ├── 选择出题数量（10/20/30/50 题）
  ├── 选择题型组合（可多选）
  └── 点击生成 → LLM 分析材料 → 返回题目列表

支持的题型：

  1. 选择题（四选一）
     ┌─────────────────────────────────────────┐
     │  Q3/20                    ⏱ 00:45      │
     ├─────────────────────────────────────────┤
     │                                         │
     │  "普惠金融"的正确英文翻译是？             │
     │                                         │
     │  ○ A. inclusive education               │
     │  ● B. inclusive finance      ← 选中     │
     │  ○ C. universal finance                 │
     │  ○ D. public finance                    │
     │                                         │

     ┌─────────────────────────────────────────┐

     ├─────────────────────────────────────────┤

     │  判断以下翻译是否正确：                   │
     │                                         │
     │  "外汇储备" → foreign exchange reserves  │
     │                                         │
     │  [ ✓ 正确 ]    [ ✗ 错误 ]               │
     │                                         │
     └─────────────────────────────────────────┘

  3. 填空题（AI 挖关键词）
     ┌─────────────────────────────────────────┐
     │  Q12/20                   ⏱ 02:10      │
     ├─────────────────────────────────────────┤
     │                                         │
     │  请填入空缺部分：                         │
     │                                         │
     │  China adheres to the principles of     │

     │                                         │

     │                                         │

     │                                         │

     │                                         │
     │  AI 评分：85/100                        │
     │  评价：回答正确，但建议补充"五个振兴"     │

     └─────────────────────────────────────────┘
```

```
AI 出题流程：
  导入材料 → 用户点击[AI 出题]
    │
    ├─ 后端将材料文本 + 用户选择的题型发送给 LLM
    ├─ LLM 返回结构化 JSON（题目 + 选项 + 正确答案 + 解析）
    ├─ 前端渲染题目列表
    │
    ├─ 用户逐题作答
    ├─ 选择题/判断题：即时判定
    ├─ 填空题：提交后比对 + 模糊匹配
    ├─ 排序题：提交后比对顺序
    └─ 简答题：提交后 LLM 评分 + 反馈
    │
    └─ 做题结束 → 统计面板
        ├─ 各题型正确率
        ├─ 薄弱知识点（LLM 分析）
        ├─ 错题自动加入错题本
        └─ [重新出题] / [只练错题]
```

```
LLM Prompt 设计要点：
  ├── 输入：材料原文 + 目标题型 + 出题数量
  ├── 约束：题目必须严格基于材料内容，不编造
  ├── 干扰项：选择题干扰项要有迷惑性（同类易混淆项）
  ├── 难度梯度：简单→中等→困难，前 30% 简单
  ├── 输出格式：严格 JSON（便于前端解析）
  └── 解析：每题附带简短解析（为什么对/为什么错）
```

#### 9.2 SM-2 间隔复习

```
复习队列：
  ├── 基于 SM-2 算法计算每张卡片的下次复习时间
```
